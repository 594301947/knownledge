线程和进程有什么区别？

其实在内核中，**线程就是进程**，即内核线程kernel_thread，它也是一个

```c
pid_t kernel_thread(int (*fn)(void *), void *arg, unsigned long flags)
{
    return _do_fork(
    		flags|CLONE_VM|CLONE_UNTRACED, 
    		(unsigned long)fn,
            (unsigned long)arg, 
            NULL, NULL, 0);
}
```
